Counter App (SwiftUI)

A simple Counter App built using SwiftUI to understand data flow between views using  
`@State` and `@Binding`.

Features
- Increment counter
- Decrement counter
- Reset counter

Learning Objectives
- Understand how `@State` stores view-local state
- Learn how `@Binding` allows child views to modify parent state
- Practice clean SwiftUI view composition

UI Enhancements
- Video background implemented using **AVKit**
- Haptic feedback on button taps:
  - Soft feedback
  - Medium feedback

Tech Stack
- **SwiftUI**
- **AVKit**

Platform
- iOS

Future Improvements
- Add animations for counter changes
- Improve accessibility
- Add unit tests



 This project is built for learning SwiftUI fundamentals and state management.
